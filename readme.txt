Synthetic-instrument-generation data collection website

Besure to use python virtual enviroment to add all the package 
framework:
This framework is run by python django. To run server in localhost, first get into python virtual enviroment and use the command "python manage.py runserver"

Database:
The database setting and client is connected with postico to enable array. To change different host, adjust setting of "DATABASE" in settings.py.
